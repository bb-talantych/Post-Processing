# Post Processing
A central repo, for all my post processing shaders and effects

This is made for Unity 6000.3.2f1 using the built in pipeline.

* **Edge Detection**
  * Depth Based
  * Depth + Normal with Noise Offsetting
* **Manga Effect**
  * Customizable Canny Edge Detector
  * Quantization of Luminance with Customizable Blending Regions
  * Customizable Hatching based on Luminance
* **Sharpness**
  * Basic Sharpening
